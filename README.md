# 📊 Sales Data Analysis Dashboard

## 📌 Project Overview
This project is a Power BI Sales Dashboard created to analyze sales performance using interactive visualizations and KPIs.

## 🚀 Features
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Sales Trend Analysis
- Category-wise Sales
- Region-wise Sales
- Interactive Filters (Slicers)

## 🛠 Tools Used
- Power BI
- Microsoft Excel
